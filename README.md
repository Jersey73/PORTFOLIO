# Manudeep Portfolio

An Apple-inspired portfolio website built with React, Vite, TailwindCSS, and Framer Motion.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and visit the local URL shown in the terminal (usually http://localhost:5173)

## Build for Production

```bash
npm run build
```

## Technologies Used

- React 18
- Vite
- TailwindCSS
- Framer Motion
- Modern JavaScript (ES6+)

## Customization

Edit `src/App.jsx` to customize:
- Your name and bio
- Project details
- Contact information
- Design and colors

The design follows Apple's minimalist aesthetic with smooth animations and clean typography.
